# compressor

compressor 
```bash

folder
tar -cvzf backup.tgz /path/to/source/folder

files
tar -cvzf backup.tgz *

extract
tar -xf backup.tgz

View
tar -ztvf backup.tar.gz

-c : Create a new archive
-v : Verbose output
-f file.tar.gz : Use archive file
-z : Filter the archive through gzip
-j : Filter the archive through bzip2

```
